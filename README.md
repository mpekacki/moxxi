# moxxi-mocks

Simple response mocker, useful for testing API callouts etc..

## Running Locally

```sh
$ npm install
$ npm run build
$ npm start
```

The app should now be running on [localhost:5000](http://localhost:5000/).

## Deploying to Heroku

```
$ heroku create
$ git push heroku main
$ heroku open
```
or
